# 内置命令

::: warning
此文档适用于 FloraBot V1.13 及以上版本
:::

内置命令是 FloraBot 自带的命令，不需要额外安装插件即可使用。

以下所有命令都需要用户ID存在于配置文件中`Administrator`字段中才能使用。

1. 重新加载所有插件:
```
/重载插件
```

2. 显示所有插件:
```
/插件列表
```

3. 启用/禁用插件:
```
/启用插件 <插件名>
/禁用插件 <插件名>
```

4. 测试命令:
```
/echo <消息>
/echo1 <消息>
```
`/echo` 和 `/echo1` 均为测试命令，可以用来测试 FloraBot 是否正常运行， 区别在于 `/echo` 属于回复消息，而 `/echo1` 属于发送消息。
