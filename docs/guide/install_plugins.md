# 安装插件
::: warning 免责声明
**声明: 插件为第三方内容, 请您自行分辨是否为恶意插件, 若被恶意插件入侵/破坏了您的设备或恶意盗取了您的信息, 造成的损失请自负, FloraBotTeam 概不负责也无义务负责!**
:::
::: tip
可以开发[插件开发](/develop/plugin_develop)自己的插件或者在[插件列表](/guide/plugin_list)寻找插件
:::
**首次启动会创建一个名为 `FloraBot` 的文件夹, 进入这个文件夹, 再进入 `Plugins` 文件夹, 将插件文件夹放入 `Plugins` 文件夹, 若是压缩了, 解压再放入即可**  
**文件结构示意:**  
```File
FloraBot
 \-Plugins
    |-插件文件夹
    |  |-Plugin.json
    |  \-插件.py
    \-插件文件夹
       |-Plugin.json
       \-插件.py
Config.json
FloraBot.py
启动脚本
```
